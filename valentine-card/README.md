# Valentine's Day Card

A beautiful interactive Valentine's Day card created with HTML, CSS, and JavaScript.

## Features

- Animated envelope opening
- Interactive Valentine's proposal
- Date planning options (Movies, Food, Custom)
- Floating hearts background
- Responsive design

## How to Use

1. Open `index.html` in your web browser
2. Click the envelope to open it
3. Follow the interactive prompts

## Hosting on GitHub Pages

This project is set up to be hosted on GitHub Pages. After pushing to a GitHub repository:

1. Go to your repository settings
2. Scroll down to "Pages" section
3. Select "Deploy from a branch"
4. Choose "main" branch and "/ (root)" folder
5. Save the settings
6. Your Valentine's card will be available at `https://yourusername.github.io/repository-name/`

## Technologies Used

- HTML5
- CSS3 (with animations)
- JavaScript (ES6+)
